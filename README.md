LM Evaluation Framework for Network Incident Diagnosis

Description:

This project implements a production-style machine learning pipeline for diagnosing simulated network incidents using telemetry data. The system integrates traditional machine learning models and transformer-based architectures and evaluates their performance using a modular evaluation framework.

Key components include:
	•	Telemetry dataset generation and preprocessing
	•	Feature engineering for network metrics
	•	Baseline ML model training using XGBoost
	•	Transformer-based classification using DistilBERT
	•	Evaluation framework computing accuracy, precision, recall, F1, and confusion matrices
	•	Experiment tracking with structured JSON logging

The project demonstrates how machine learning and transformer models can be evaluated and benchmarked for network fault diagnosis tasks similar to those used in digital experience monitoring platforms.
